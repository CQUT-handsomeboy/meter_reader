# YOLO_thermometer_read

## 思路与步骤

1.  目标检测，切割出仪表ROI

2.  对仪表ROI再次使用目标检测，得到三个ROI，取三个ROI中心坐标构成一组关键点矩阵

3.  使用上述关键点矩阵与标准图像关键点矩阵实现仿射变换，得到倾斜校正以后的仪表盘

4.  对倾斜校正以后的仪表盘进行实例分割，得到表针的中心坐标

5.  表针的中心坐标与标准图像的表针的起点坐标构成表针向量，与标准图像的表针向量求夹角，**注意求夹角要注意处理旋转角度超过180度的情况，因此必须规定方向是顺时针还是逆时针**。

## blob

blob中的是三个权重文件，ultralytics版YOLOv8n，Pytorch权重文件格式。
